# Fika Weapon Pickup Fix

**Deployment:** `(client_only)`

**Version:** `2.0.0`

**Platform:** EFT 16.9 / SPT 4.0.13 / Fika 2.3.x

Убирает зависших FirearmController'ов у мёртвых ботов, чьё оружие уже подобрал игрок. Работает по таймеру (каждые 3 сек), не патчит Proceed — нулевое вмешательство в систему экипировки оружия Fika.

## Install

1. Download release zip from GitHub Releases.
2. Extract into Escape from Tarkov game root (`EscapeFromTarkov.exe`).
3. See `INSTALL.md` in release folder (not inside zip).
